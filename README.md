**To set up and run the tests make the next steps**:
1) Copy the existing project
2) Install Cypress using next command:

   ```npm install cypress --save-dev```
   
   or
   
   ```yarn add cypress --dev```
   
4) Run Cypress tests:

   ```./node_modules/.bin/cypress run```
   
   or
   
   ```npx cypress run```
